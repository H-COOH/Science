# 社会科学学习月

!!! success "结果"

	本活动正在进行中。

!!! info "时间"

	2025年10月01日 — 2025年10月31日

!!! quote "背景"

	我们曾试图从社会科学中借鉴关于个人成长的经验，并将每年的10月设立为社会科学学习月。尽管我们的生产生活秩序已经趋于稳定，我们仍选择保留这一传统，以此为契机促进人的全面发展。

!!! tip "主题"

	本年度的主题为：同大江东去
	
	> 乱石穿空，惊涛拍岸，卷起千堆雪。

!!! abstract "目标"
	
	在本年度的社会科学学习月中，我们将以“大语文”为关键词，品味语言艺术，领略诗词之美，积累文化底蕴，提升人文修养，丰富自己的精神世界。
