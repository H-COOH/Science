# 心理健康建设周

!!! failure "结果"

	本活动因时间冲突而被取消。

!!! info "时间"

	2025年5月24日 — 2025年5月30日

!!! quote "背景"

	为纪念 *未命名事件*，响应 5·25 心理健康节，发扬《策略总纲》第三条的精神，我们每年将5月25日所在的星期设立为“心理健康建设周”，旨在鼓励认识自我、理解自我、接纳自我，进而推动心理健康建设。

!!! tip "主题"

	本年度的主题为：探索、理解、宽容

!!! abstract "目标"

	在本年度的心理健康建设周中，我们将以“恐惧感”为主线，补充生物学和心理学的基本常识，学习情绪的基本原理，揭秘我们内心深处的恐惧，坚定成长与向前的信心。
